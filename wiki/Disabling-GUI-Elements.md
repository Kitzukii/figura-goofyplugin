# Disabling GUI Elements

To disable a GUI element do the following:
```lua
goofy:setDisableGUIElement(element, true)
```
to re-enable an element, run the same command with `false` instead of true

## Disablable GUI elements
* "HOTBAR"
* "JUMP_METER"
* "EXPERIENCE_BAR"
* "SELECTED_ITEM_NAME"
* "SCOREBOARD_SIDEBAR"
* "PLAYER_HEALTH"
* "VEHICLE_HEALTH"
* "TEXTURE_OVERLAY"
* "SPYGLASS_OVERLAY"
* "VIGNETTE"
* "PORTAL_OVERLAY"
* "CHAT"
* "BOSSBAR"
* "TAB_LIST"
